# DropShoes Style Guide

Este guia define os padrões visuais e componentes para a aplicação DropShoes, garantindo consistência em toda a interface do usuário.

## Cores

### Paleta Principal - Zinc

Nossa paleta principal é baseada nos tons de Zinc do Tailwind CSS, proporcionando uma base neutra e elegante.

| Nome     | Classe Tailwind | Hex       | Uso                                            |
| -------- | --------------- | --------- | ---------------------------------------------- |
| Branco   | `bg-white`      | `#ffffff` | Fundo de cards, áreas de conteúdo              |
| Zinc-50  | `bg-zinc-50`    | `#fafafa` | Fundos alternativos, hover de botões secondary |
| Zinc-100 | `bg-zinc-100`   | `#f4f4f5` | Bordas leves, backgrounds de inputs            |
| Zinc-200 | `bg-zinc-200`   | `#e4e4e7` | Separadores, bordas                            |
| Zinc-300 | `bg-zinc-300`   | `#d4d4d8` | Ícones desabilitados, textos terciários        |
| Zinc-400 | `bg-zinc-400`   | `#a1a1aa` | Texto desabilitado, placeholders               |
| Zinc-500 | `bg-zinc-500`   | `#71717a` | Textos secundários, ícones                     |
| Zinc-600 | `bg-zinc-600`   | `#52525b` | Texto secundário em fundos claros              |
| Zinc-700 | `bg-zinc-700`   | `#3f3f46` | Texto de destaque secundário                   |
| Zinc-800 | `bg-zinc-800`   | `#27272a` | Headers secundários, texto em fundos claros    |
| Zinc-900 | `bg-zinc-900`   | `#18181b` | Texto principal em fundos claros               |
| Zinc-950 | `bg-zinc-950`   | `#09090b` | Headers, texto em destaque, texto principal    |

## Tipografia

### Família da Fonte

Usar a fonte padrão do sistema para melhor performance:

```css
font-family: Arial, Helvetica, sans-serif;
```

### Tamanhos de Fonte

| Nome | Classe Tailwind | Uso                                 |
| ---- | --------------- | ----------------------------------- |
| XS   | `text-xs`       | Legendas, notas de rodapé           |
| SM   | `text-sm`       | Texto secundário, descrições        |
| Base | `text-base`     | Texto padrão do corpo               |
| LG   | `text-lg`       | Subtítulos, texto enfatizado        |
| XL   | `text-xl`       | Títulos de cards, nomes de produtos |

| 3XL  | `text-3xl`      | Títulos de páginas                  |
| 4XL  | `text-4xl`      | Banners, headlines                  |

### Peso da Fonte

| -------- | --------------- | ------------------------------------ |

## Componentes Principais

### Botões

#### Botão Primário

```jsx
<button className="bg-zinc-950 text-white px-4 py-2 rounded-md hover:bg-zinc-900 transition-colors">
  Comprar Agora
</button>
```

#### Botão Secundário

```jsx
<button className="bg-white text-zinc-950 border border-zinc-200 px-4 py-2 rounded-md hover:bg-zinc-50 transition-colors">
  Ver Detalhes
</button>
```

#### Botão de Texto

```jsx
<button className="text-zinc-700 hover:text-zinc-950 font-medium transition-colors">
  Ver Mais
</button>
```

### Cards

#### Card de Produto

```jsx
<div className="bg-white rounded-lg shadow p-4 hover:shadow-md transition-shadow">
  <div className="relative aspect-square mb-4">
    <Image
      src="/product.jpg"
      alt="Product"
      fill
      className="object-cover rounded-md"
    />
  </div>
  <h3 className="text-xl font-semibold text-zinc-950">Nome do Produto</h3>
  <p className="text-zinc-600 mt-1">Descrição curta do produto</p>
  <div className="mt-2 flex justify-between items-center">
    <span className="text-lg font-bold text-zinc-950">R$ 299,90</span>
    <button className="bg-zinc-950 text-white p-2 rounded-md">
      <ShoppingCart size={18} />
    </button>
  </div>
</div>
```

### Formulários

#### Input de Texto

```jsx
<div className="mb-4">
  <label
    htmlFor="name"
    className="block text-sm font-medium text-zinc-700 mb-1"
  >
    Nome Completo
  </label>
  <input
    type="text"
    id="name"
    className="w-full px-3 py-2 border border-zinc-200 rounded-md focus:outline-none focus:ring-2 focus:ring-zinc-500 focus:border-transparent"
    placeholder="Digite seu nome"
  />
</div>
```

#### Select

```jsx
<div className="mb-4">

```

## Tokens de Design

Para facilitar a manutenção, definimos os seguintes tokens de design no arquivo globals.css:

```css
:root {
  --background: #ffffff;
  --foreground: #171717;
}

@media (prefers-color-scheme: dark) {
  :root {
    --background: #0a0a0a;
    --foreground: #ededed;
  }
}
```

Estes tokens são utilizados por toda a aplicação para garantir consistência.
